INFOVIS - individual project

This simple project shows how to visualize some data from a json file using d3js.

The json file contains 10 data-cases with 5 variables (x, y, base, height, color), each isosceles triangle is a datacase. The triangle is drawn as a path that starts from (x,y), than it moves to the top corner, to the right corner and lastly it closes itself. The user can interact with the figures and they will show a cool animation.

Interaction: Click on one triangle to select it. Click on a second triangle to switch the base and the height of them.

Feature: Fully responsive to window resizing, interactive with fluid animations and eye-saving pastel colors.
